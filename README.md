# Used Furniture Sales Application 
**Undergraduate Thesis**

A full-stack marketplace platform for buying and selling second-hand furniture, built with Spring Boot, React, and MySQL.

## Features & Requirements

### For Users (Role: USER)
* **Authentication:** Registration and login system.
* **Profile Management:** Edit personal profile and view seller profiles.
* **Ad Management:** Create, edit, and delete furniture listings.
* **Browsing:** View all listings with detailed information and filtering.
* **Orders:** Purchase products and manage personal order history.
* **Favorites:** Save preferred listings for quick access.

### For Administrators (Role: ADMIN)
* **User Management:** View all registered users and toggle account status (activate/deactivate).
* **Content Moderation:** Full access to view, manage, and delete any listing or order across the platform.
* **System Oversight:** Monitor all transactions and user activity.

### Non-Functional Requirements
* **Security:** Passwords are encrypted before being stored in the database to ensure data privacy.
* **Usability:** Simple and intuitive User Interface (UI) designed for a seamless user experience.



## Tech Stack
* **Backend:** Java Spring Boot (REST API)
* **Frontend:** React.js & CSS
* **Database:** MySQL
* **Security:** Bcrypt Password Encoding
* **Build Tool:** Maven
